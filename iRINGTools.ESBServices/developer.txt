==Java Development Environment Configuration==

1. Do SVN checkout/update

2. Start Eclipse

3. Import user libraries 
   a. Make copy iringtools.default.userlibraries and replace library paths to match your "libraries" directory path.
   b. Window - Preferences - Java - Build Path - User Libraries - <your copy of iringtools userlibraries>

4. Delete projects (without deleting the contents) if exist


5. Import projects into workspace
   File - Import - General - Existing Projects into Workspace - select iRINGTools.ESBServices root folder

6. Create Tomcat server instance in eclipse if not exist
   Right click an empty spot in Servers tab - New - Server - Apache - Tomcat 6.0 - Next - select tomcat home directory
