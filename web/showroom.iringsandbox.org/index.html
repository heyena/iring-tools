Showroom

The showroom is place where you can kick the tires on iRINGTools. There are test applications (EM and MM) in a test project. You can search and browse the reference data. Try mapping from a test application to the referecne data and try our RESTful services or SPARQL endpoints for reviewing the data represented in various ISO 15926 formats (part7, part8 or part9).
