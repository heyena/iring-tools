==Java Development Environment Configuration==
 * Do SVN checkout/update
 * Start Eclipse
 * Import user libraries 
  * Make copy iringtools.default.userlibraries and replace library paths to match your "libraries" directory path.
  * Window - Preferences - Java - Build Path - User Libraries - <your copy of iringtools userlibraries>
 * Delete projects (without deleting the contents)
 * Import projects into workspace
  * File - Import - General - Existing Projects into Workspace - select root(workspace) folder
 * Create Tomcat server instance in eclipse
  * Right click an empty spot in Servers tab - New - Server - Apache - Tomcat 6.0 - Next - 
    select tomcat home directory
