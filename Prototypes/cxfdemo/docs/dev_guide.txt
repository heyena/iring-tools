Apache CXF web services in Eclipse

- prerequisites:
  - download and install java jdk (v6)
      http://java.sun.com/javase/downloads/index.jsp
  - download and extract eclipse EE (gallieo) 
      http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/galileo/SR2/eclipse-jee-galileo-SR2-win32.zip
  - download and install apache tomcat (v6)
      http://tomcat.apache.org/download-60.cgi
  - download apache cxf (v2.2.9) and extract it to <tomcat home>/shared folder
      http://mirror.cloudera.com/apache/cxf/2.2.9/apache-cxf-2.2.9.zip
  - download and extract jsr311-api
      http://www.java2s.com/Code/Jar/STUVWXYZ/Downloadjsr311apijar.htm
  
- configure eclipse:
  - start up eclipse
  - add tomcat server to eclipse
    - show Servers view
      click on Windows on menu bar - Show View - Servers
    - add server
      - click Servers tab in the bottom pane - right click on an empty spot in the Servers pane - New - Server - Tomcat v6.0 Server
      - follow the wizard (tomcat home folder is C:\Program Files\Apache Software Foundation\Tomcat 6.0 by default)
  - configure tomcat to use load cxf library at run-time
    - expand Servers item on the left pane - Tomcat v6.0 Server - catalina.properties
    - edit shared.loader property as follows:
        shared.loader=${catalina.base}/shared/apache-cxf-2.2.9/lib,${catalina.base}/shared/apache-cxf-2.2.9/lib/*.jar
  - add user libraries
    - add cxf library
      - click on Windows on the menu bar - Preferences - Java - Build Path - User Libraries - New
      - enter library name
      - click on Add JARs - browse to <cxf home>/lib and select all the jar files
      - OK - Finish
    - similarly add jsr311-api library
    
- import project into workspace
    - File - Import - General - Existing Projects into Workspace - Next
    - follow the wizard
  
- access the web service:
    http://<server>[:port]/<project name>/<service endpoint>/<path>/[<param1>/<param2>...]
    eg.: http://localhost:8080/cxfdemo/EmployeeService/get
         http://localhost:8080/cxfdemo/EmployeeService/echo/Hahn/Le
